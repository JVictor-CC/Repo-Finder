## Features

* Responsive Design
* displays profile info (Name, Bio, Picture)
* Displays repository info (Name, Description, Topics, Most used languages...)
* Link to the repository
* Link to each topic
* Error messages
* Disabled submit button while validation fails

## Built With

* React
* Javascript
* Github API
